## 주요 기능
- webRTC를 통한 실시간 화상 통역 매칭 서비스
- STT를 이용한 대화 내용 인식 및 기록
- AI를 이용한 수어 to 자막 통역 서비스
- AI를 이용한 음성 to 애니메이션 통역 서비스
