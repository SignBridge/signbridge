# 시연 시나리오
## 기능 별 설명
### 메인
![Image_Pasted_at_2023-2-16_14-58](/uploads/ac39e60aa68b82d93ea2f88a5ac9c18a/Image_Pasted_at_2023-2-16_14-58.png)
#### 1. 통역사 매칭 서비스
통역사와 매칭 서비스를 이용할 수 있는 버튼. 통역사가 수락하기 전까지 매칭 대기화면에서 대기하게 됩니다.
#### 2. AI 통역 서비스
AI 통역 기능을 이용할 수 있는 버튼. 대기없이 바로 사용 가능합니다.
#### 3. 소개 영상
해당 사이트에 대한 간단한 설명과 메인 기능에 대한 설명을 수어 영상으로 보여주고 있습니다.
#### 4. 사이트 이름 및 로고
클릭 시 메인 화면으로 이동 가능합니다.
#### 5. 통역사 로그인 버튼
통역사용 로그인 화면으로 가는 버튼입니다. 통역사는 로그인 이후 통역 수락 가능합니다.
